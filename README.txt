This is a WORK IN PROGRESS mod for the game Minecraft in which I program as many Magical Items from the popular TableTop genre, Dungeons & Dragons, as I can.

These magical items are all being referenced from Wimblerock's Auction catalog, that can be found here: https://www.gmbinder.com/share/-Lmg2hIXBKOY9PlGQRe_

COMPLETED ITEMS:
Maul of the Mule
Explorer's Cutlass
Combat Bandage
Slip Razor
Shield of the Tower
Chime of Beast Discovery
